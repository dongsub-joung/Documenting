## HTPP

통신 기술, 가장 중요한 프로토콜

### Rquest message

- request message header
  - Host //개개인의 컴퓨터
  - User-Agent: 접속자의 물리,소프트웨어 정보
  - Accept-Encoding: 압축 방식의 지원 포멧
  - If-Modified-Since: 마지막 업뎃 날짜

- Response Header

- 성능향상

  cache-control

  pragma

- 인식

  로그인 기록, cookie 값 설정, 전송, 식별

  web storage

- proxy

- wireshark.org //트래픽 감시 기술
